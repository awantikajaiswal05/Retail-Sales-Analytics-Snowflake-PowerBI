# 📊 Retail Sales Analytics Dashboard using Snowflake, SQL & Power BI

## 📌 Project Overview

This project demonstrates an end-to-end Retail Sales Analytics solution using **Snowflake Cloud Data Warehouse**, **SQL**, and **Microsoft Power BI**. The retail sales dataset was loaded into Snowflake, analyzed using SQL queries, and visualized through interactive Power BI dashboards to provide meaningful business insights.

---

## 🎯 Project Objectives

- Analyze retail sales performance
- Identify top-performing and loss-making products
- Monitor KPIs such as Sales, Profit, Orders, and Customers
- Build interactive dashboards for business decision-making
- Demonstrate Snowflake and Power BI integration
