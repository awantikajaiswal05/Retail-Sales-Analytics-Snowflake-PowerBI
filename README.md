# 📊 Retail Sales Analytics Dashboard using Snowflake & Power BI

## 📌 Project Overview

This project demonstrates an end-to-end Retail Sales Analytics solution using Snowflake Cloud Data Warehouse, SQL, and Microsoft Power BI. The retail sales dataset was loaded into Snowflake, analyzed using SQL queries, and visualized through interactive Power BI dashboards to generate actionable business insights.

## 🎯 Objectives

- Analyze retail sales performance
- Identify top-performing and loss-making products
- Track sales, profit, and customer trends
- Build interactive dashboards for decision-making
- Demonstrate Snowflake and Power BI integration

## 🛠️ Tech Stack

- Snowflake
- SQL
- Microsoft Power BI

## 📊 Dashboard Features

- Executive Dashboard
- Sales Analysis Dashboard
- KPI Cards
- Interactive Slicers
- Sales by Category
- Top 10 Products
- Profit Analysis
- Regional Performance Analysis
